## 3. 핵심 미구현 기능 (기획서 대비)

### P0 — 핵심 기능

| 기능 | 설명 | 관련 기획 |
|------|------|----------|
| **실시간 AI 아젠다 추출** | 미팅 중 대화를 분석하여 아젠다 항목 자동 생성/업데이트. 현재는 데모 데이터만 존재, 실시간 AI 파이프라인 필요 | 별첨6 - 실시간 요약 |
| **온디맨드 힌트 버튼** | 사용자가 힌트 필요 시 명시적으로 클릭하여 즉시 AI 코칭 받는 기능 | 별첨6 - 온디맨드 힌트 |
| **화자 분리 매핑** | CLOVA API 연동, 미팅 종료 후 화자(A=고객, B=나) 프로필 카드 제안 및 스크립트 일괄 적용 | 별첨6 - 화자 분리 |
| **모듈형 회의록 프리셋** | B2B 추천 프리셋 기반 맞춤형 회의록 양식. 사용자가 목차 구성 및 프롬프트 지침 입력 | 별첨6 - 모듈형 회의록 |

### P1 — 필수 기능

| 기능 | 설명 | 관련 기획 |
|------|------|----------|
| **Google Calendar 연동** | 실제 캘린더 API 연동 → 미니 캘린더 + 다가오는 미팅 자동 표시, 미팅 정보 사전 주입 | 별첨6 - 캘린더 연동 |
| **미팅 DB 저장/조회 완성** | `/api/meeting` 구현, 미팅 기록 CRUD, 대시보드 미팅 리스트 실제 데이터 연동 | 별첨6 - 대시보드 |
| **프로젝트 ↔ 미팅 오토 라우팅** | 캘린더 고객사 정보 기반 미팅 → 프로젝트 폴더 자동 분류 | 별첨2 §4 |
| **파일 내보내기 확장** | PDF, Word(.docx), 마크다운(.md) 다운로드 지원. 현재 TXT + 클립보드만 존재 | 별첨2 §3 |
| **AI 단어 추천** | 미팅 종료 후 빈도수 높은 미등록 명사를 사전 추가 추천 리스트로 제공 | 별첨2 §2 |
| **내 프로필 설정 페이지** | 소속, 직무, 담당 제품, 영업 스타일 설정 → AI 힌트 개인화 | 별첨3 §3 |
| **설정 페이지** | 알림, 언어, AI 힌트 빈도, 클라이언트 모드 단축키 등 설정 | 별첨6 |

### P2 — 확장 기능

| 기능 | 설명 | 관련 기획 |
|------|------|----------|
| **결제/구독 시스템** | Stripe 등 연동, Free/Basic/Pro 플랜 전환, 사용량 과금 | 별첨4 |
| **지식 베이스 업로드 (RAG)** | PDF/PPTX/DOCX 업로드 → AI 청킹 → 벡터 DB 저장 → 실시간 미팅 중 검색 | 별첨3 §1-2 |
| **팀 대시보드** | 매니저가 팀원 미팅 현황 통합 조회 (고객 인터뷰에서 요청 빈도 1위) | 고객 피드백 |
| **CRM 연동** | Salesforce 자동 동기화 (고객 인터뷰에서 요청 빈도 2위) | 고객 피드백 |
| **다국어 지원** | 한영 혼합 미팅 STT 인식률 개선 (고객 인터뷰에서 요청 3위) | 고객 피드백 |
| **모바일 앱 (Capacitor)** | 폰에서 녹음 → PC에서 화면 보기 크로스디바이스 실시간 동기화 | 별첨1 §1 |
| **프로젝트별 지식 지정** | 미팅 시작 전 사용할 지식(제안서 A, 소개서 B) 선택 | 별첨3 §3 |

## 4. 기술 부채 / 개선 사항

| 항목 | 설명 |
|------|------|
| **Silero VAD 클라이언트 적용** | 현재 모든 오디오를 서버로 전송. VAD 적용 시 STT 비용 30~40% 절감 가능 |
| **CLOVA Speech 사후 처리 파이프라인** | 미팅 종료 후 전체 녹음 파일을 CLOVA API로 돌려 고정밀 화자 분리 + IT 용어 인식 |
| **웹소켓 실시간 동기화** | 현재 HTTP 기반 → WebSocket으로 전환하여 다중 디바이스/다중 탭 실시간 동기화 |
| **보안 강화** | Row Level Security 전수 점검, Zero Data Retention API 설정, 데이터 격리 검증 |
| **에러 핸들링 고도화** | 네트워크 단절/STT 타임아웃/API 장애 시 graceful fallback |
