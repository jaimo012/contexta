**중요도 기준**
- HIGH: 서비스의 본질적 가치를 제공하기 위해 필수적

- LOW: 고객 편의 및 가치 증대에 기여할 수 있음

**진행상태**: 대기 / 개발완료 / 보류

## 3. 핵심 미구현 기능

### HIGH — 서비스 본질 가치

| 기능 | 설명 | 관련 기획 | 진행상태 |
|------|------|----------|---------|
| **다운로드** | 회의 종료 후 녹음파일, 스크립트, 메모를 다운받을 수 있다. PDF, Word(.docx), 마크다운(.md) 지원. 현재 TXT + 클립보드만 존재 | 별첨2 §3 | 대기 |
| **회의록 양식 (모듈형)** | 회의록 양식을 사용자가 직접 설정. 모듈형으로 주고 사용자가 조립. B2B 추천 프리셋 기반 맞춤형 양식, 프롬프트 지침 입력 | 별첨6 - 모듈형 회의록 | 대기 |
| **실시간 AI 아젠다 추출** | 미팅 중 대화를 분석하여 아젠다 항목 자동 생성/업데이트. 현재는 데모 데이터만 존재, 실시간 AI 파이프라인 필요 | 별첨6 - 실시간 요약 | 대기 |
| **온디맨드 힌트 버튼** | 사용자가 힌트 필요 시 명시적으로 클릭하여 즉시 AI 코칭 받는 기능 | 별첨6 - 온디맨드 힌트 | 대기 |
| **화자 분리 매핑** | CLOVA API 연동, 미팅 종료 후 화자(A=고객, B=나) 프로필 카드 제안 및 스크립트 일괄 적용 | 별첨6 - 화자 분리 | 대기 |

### MID — 고객 편의 핵심

| 기능 | 설명 | 관련 기획 | 진행상태 |
|------|------|----------|---------|
| **캘린더 연동** | 자신의 구글 캘린더를 연동(CRUD). 미니 캘린더 + 다가오는 미팅 자동 표시, 미팅 정보 사전 주입 | 별첨6 - 캘린더 연동 | 대기 |
| **회의록 수정** | AI가 작성한 회의록을 사용자가 직접 수정할 수 있다 | 별첨6 - 회의록 | 대기 |
| **미팅 DB 저장/조회 완성** | `/api/meeting` 구현, 미팅 기록 CRUD, 대시보드 미팅 리스트 실제 데이터 연동 | 별첨6 - 대시보드 | 대기 |
| **사전학습 (RAG)** | 유저가 자신 회사의 소개자료나 제안서 등의 자료를 넣을 수 있고, 그 지식을 바탕으로 작동한다. PDF/PPTX/DOCX 업로드 → AI 청킹 → 벡터 DB 저장 → 실시간 미팅 중 검색 | 별첨3 §1-2 | 대기 |
| **AI 단어 추천** | 미팅 종료 후 빈도수 높은 미등록 명사를 사전 추가 추천 리스트로 제공 | 별첨2 §2 | 대기 |
| **내 프로필 설정 페이지** | 소속, 직무, 담당 제품, 영업 스타일 설정 → AI 힌트 개인화 | 별첨3 §3 | 대기 |
| **설정 페이지** | 알림, 언어, AI 힌트 빈도, 클라이언트 모드 단축키 등 설정 | 별첨6 | 대기 |

| **모바일 앱 (Capacitor)** | 폰에서 녹음 → PC에서 화면 보기 크로스디바이스 실시간 동기화 | 별첨1 §1 | 대기 |
| **프로젝트별 지식 지정** | 미팅 시작 전 사용할 지식(제안서 A, 소개서 B) 선택 | 별첨3 §3 | 대기 |

## 4. 기술 부채 / 개선 사항

| 항목 | 설명 | 중요도 | 진행상태 |
|------|------|--------|---------|
| **Silero VAD 클라이언트 적용** | 현재 모든 오디오를 서버로 전송. VAD 적용 시 STT 비용 30~40% 절감 가능 | HIGH | 대기 |
| **CLOVA Speech 사후 처리 파이프라인** | 미팅 종료 후 전체 녹음 파일을 CLOVA API로 돌려 고정밀 화자 분리 + IT 용어 인식 | HIGH | 대기 |
| **웹소켓 실시간 동기화** | 현재 HTTP 기반 → WebSocket으로 전환하여 다중 디바이스/다중 탭 실시간 동기화 | MID | 대기 |
| **보안 강화** | Row Level Security 전수 점검, Zero Data Retention API 설정, 데이터 격리 검증 | MID | 대기 |
| **에러 핸들링 고도화** | 네트워크 단절/STT 타임아웃/API 장애 시 graceful fallback | MID | 대기 |
